# Phase 5 - AI Voice Ordering (Mobile)

This folder adds a lightweight mobile-app implementation for AI voice ordering without paid services.

## Included features
- Popup to choose **AI Voice Mode** or **Manual Mode**
- Speech-to-text integration placeholder (`startSpeechToText`)
- Text-to-speech placeholder (`speakText`)
- Simple keyword-based NLP command parsing
- Voice-based add-to-cart
- Voice-based order confirmation
- Optional API key file (`src/config/ai.ts`) kept empty by default

## Notes
- Current speech functions are intentionally placeholder-only and free-library friendly.
- No paid API is required for this implementation.
